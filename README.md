# Python Network Scanner

A simple Python network scanner that detects open ports on a target machine.

## Features

- Scan a target IP address
- Detect open ports
- Show basic port status

## Technologies

- Python
- Networking (TCP sockets)

## Usage

Run the scanner:
```
python scanner.py
```

Enter the target IP address when prompted.

Example:

```
Enter target IP: 192.168.1.1

Scanning target 192.168.1.1

[OPEN] Port 80
[CLOSED] Port 21
[CLOSED] Port 22
```

## Learning Purpose

This project demonstrates:

- Python scripting
- Socket programming
- Networking fundamentals
- TCP port scanning
